== Installation ==


1. Make sure you have WooCommerce installed on your WordPress website
2. Upload the downloaded woocommerce-livechat file to WordPress plugins directory (/wp-content/plugins/)
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Navigate to the "WooCommerce" menu on the left and select the "LiveChat" section
5. Create a new LiveChat account, or use your existing one
6. Choose which custom variables will be passed to LiveChat and get ready to chat

== Frequently Asked Questions ==

__What limitations do trial accounts have?__
The trial offers all features of the Team plan. During a trial period, you can give LiveChat access up to 100 users. If you need more, or if you need to try out LiveChat in another plan, we can also manage it for you.

__How does LiveChat work when no one is online?__
When users are not logged in, the chat window turns into the ticket form. Customers can leave messages as tickets and contact you after work hours. 

__Is there any option to customize the appearance of the chat window?__
The chat window can be fully customized. You can choose from several chat window themes, upload your company's logo and change the color of the chat window to fit your branding.

__Can one user respond to multiple chats at the same time?__
Each user can chat with up to 99 customers simultaneously. 

__Is it possible to add LiveChat to more than just one website?__
You can use one account to embed LiveChat on as many websites you like and each one can have its own, unique design. 

__Does LiveChat have mobile applications?__
LiveChat offers multiple mobile applications for iPhone, iPad and Android devices. [Learn more](https://www.livechatinc.com/kb/stay-mobile-with-livechat/?a=woocommerce&utm_source=woocommerceplugin&utm_medium=integration&utm_campaign=woocommerceplugin "Learn more").

__Which languages does LiveChat support?__
LiveChat is available in 41 languages including: Chinese, English, French, German, Indonesian, Portuguese and Spanish. The list of available languages can be found [here](https://www.livechatinc.com/kb/how-to-modify-chat-window-language/?a=woocommerce&utm_source=woocommerceplugin&utm_medium=integration&utm_campaign=woocommerceplugin "LiveChat Knowledge Base, Support").
